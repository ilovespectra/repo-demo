![url](https://media.discordapp.net/attachments/1051281685234327613/1122565749257547989/Screenshot_2023-06-25_at_10.29.10_AM.png?width=1554&height=650)

![url2](https://media.discordapp.net/attachments/1051281685234327613/1122565749622448279/Screenshot_2023-06-25_at_10.30.06_AM.png?width=1864&height=602)
